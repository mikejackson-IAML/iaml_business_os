# 06-03: Dashboard UI - Show Viewed Status in Not Responded List

## Objective
Update the dashboard to display "Viewed" badges on instructors who have opened their notification, with a tooltip showing the timestamp. The not responded list should sort Not Viewed instructors first.

## Context
The `not_responded_instructors` view (updated in 06-01) now includes `viewed_at`. We need to:
1. Update TypeScript types to include `viewed_at`
2. Update the query function (view already handles sorting)
3. Add "Viewed" badge with tooltip to the NotRespondedList component

Per context decisions:
- Subtle 'Viewed' text badge (neutral/gray color)
- Badge placement in status area with other badges
- Timestamp shown on hover/tooltip only: "Viewed Jan 22 at 3:45 PM"
- Not Viewed is higher priority (no badge shown)
- Sort order handled by view (Not Viewed first)

    month: 'short',
    day: 'numeric',
    hour: 'numeric',

  });
}
