## Objective

Initialize a Next.js 14+ application with App Router, shadcn/ui component library, and Supabase client for the instructor portal at faculty.iaml.com.

## Context

- This is a standalone app, not part of the main website
- Uses Next.js App Router (not Pages Router)
- shadcn/ui provides accessible, customizable components built on Radix primitives
- Supabase client needed for database queries and RPC calls
- Will be deployed to Vercel at faculty.iaml.com subdomain

<task id="5">
Create Supabase client utility at `faculty-portal/lib/supabase.ts`:
```typescript
import { createClient } from '@supabase/supabase-js'

const supabaseUrl = process.env.NEXT_PUBLIC_SUPABASE_URL!
const supabaseAnonKey = process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!

export const supabase = createClient(supabaseUrl, supabaseAnonKey)

// Server-side client with service role for admin operations
export function createServerClient() {
  return createClient(
    process.env.NEXT_PUBLIC_SUPABASE_URL!,
    process.env.SUPABASE_SERVICE_ROLE_KEY!
  )
}
```
</task>
