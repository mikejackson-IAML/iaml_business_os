# Milestone v1.0: Faculty Program Scheduler MVP

**Status:** SHIPPED 2026-01-22
**Phases:** 1-5
**Total Plans:** 17

## Overview

Build the complete Faculty Program Scheduler from foundation to dashboard integration. A tiered instructor assignment system that automatically notifies qualified faculty about available teaching opportunities, respects priority relationships (VIP -> Local -> Open), and provides IAML with full visibility into the instructor recruitment pipeline.

## Milestone Summary

**Key Decisions:**
- Next.js 14+ App Router + shadcn/ui for frontend
- Magic link authentication (no passwords)
- 3-tier system: VIP (7d) -> Local (5d) -> Open
- Local definition: Same state as program location
- Instant first-come-first-served claims (no approval workflow)

**Issues Resolved:**
- SendGrid nodes require explicit Sender Name and Message Body after import
- After action nodes, use `$('SourceNode').item.json.*` to access original data

**Tech Debt Incurred:**
- Workflows registered in n8n_brain (tracked, script created)
- sendNudge webhook fallback added (fixed)
- Phases 1-3 missing formal VERIFICATION.md files (documentation only)
- RPC calls in faculty-portal could use explicit schema prefix (functional)
