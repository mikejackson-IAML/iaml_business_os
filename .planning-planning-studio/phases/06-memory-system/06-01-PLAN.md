- Define `MemoryType = 'decision' | 'inspiration' | 'insight' | 'pivot' | 'research_finding' | 'constraint' | 'user_preference' | 'rejection_reason'`
- Define `ExtractedMemory = { content: string; memory_type: MemoryType; source_phase?: string }`
- Export `extractMemories(conversationText: string, projectName: string, phaseType: string): Promise<ExtractedMemory[]>`
  - Calls Claude (claude-sonnet-4-20250514) with tool_use to get structured JSON
  - Define a tool `extract_memories` with parameters: `memories: Array<{ content: string, memory_type: MemoryType }>`
  - System prompt: "Extract notable memories from this planning conversation.   - Set source_phase from phaseType parameter

- After insert, fire-and-forget embedding generation: for each memory, call generateEmbedding(memory.content), then update the row with `embedding: JSON.stringify(embeddingArray)` — CRITICAL: must use JSON.stringify for pgvector serialization via Supabase client

- Return `{ stored: number, embedded: number }` counts
