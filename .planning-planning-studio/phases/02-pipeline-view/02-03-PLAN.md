# Phase 02, Plan 03: Quick Capture Modal

## Objective

Add a "+ Capture Idea" button in the page header that opens a modal for quickly capturing new project ideas with just a title and optional one-liner.

Purpose: Zero-friction idea capture is critical for a planning tool. Users should be able to capture a thought in seconds without leaving the pipeline view.

Output: Header button + modal form that creates new projects in the Idea column.

- Close on overlay click: add `onClick` on overlay div, check `e.target === e.currentTarget`

## Verification

1. "Capture Idea" button visible in page header
2. Click button — modal opens with focus on title field
3. Submit without title — shows validation error
4. Enter title + one-liner, submit — modal closes, toast shows "Idea captured!"
5. New card appears in Idea column
6. Click overlay or press Escape — modal closes without creating
7. Reopen modal — form fields are empty (reset)

## Success Criteria

- Capture button prominently visible in header
- Modal follows existing codebase modal pattern (fixed overlay, Card body)
- Title required, one-liner optional
- Creates project with status='idea', current_phase='capture'
- Toast feedback on success/failure
- Form resets on close
