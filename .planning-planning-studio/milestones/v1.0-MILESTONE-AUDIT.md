## Resolved Gaps

### Phase 7 Document Version Response (FIXED)

**Original Issue:** `saveDocumentVersion` returned `{id, version}` but API routes accessed `saved.doc_type` and `saved.content` which were undefined.

**Resolution:** The insert now includes `.select('id, version, doc_type, content')` and returns the full object. Both generate and edit routes work correctly.
