## Feature Specifications

### [x] F1: Pipeline View (Main Dashboard) — v1.0

Kanban-style view of all ideas across the pipeline with drag-and-drop, search/filter, and quick capture.

**Outcome:** Fully implemented in Phase 2. Columns: Incubating, Active, Ready to Build, Building, Shipped.

### [x] F2: Project Detail View — v1.0

Deep dive with conversation interface, phase progress, sidebar panels.

**Outcome:** Implemented in Phase 3. Left sidebar with sessions/documents/research, main chat area.

### [x] F3: Incubation State — v1.0

Special UI when project is locked for incubation.

**Outcome:** Implemented in Phases 3 and 5. Countdown timer, idea capture textarea, skip with confirmation.

### [x] F4: Ready-to-Build Queue — v1.0

Prioritized list of projects ready for development.

**Outcome:** Implemented in Phase 9. AI-prioritized with Claude scoring, pinning, reasoning display.

### [x] F5: Build Tracker — v1.0

Track active development progress.

**Outcome:** Implemented in Phase 10. Progress stepper, shipping flow, status-specific card displays.

### [x] F6: Global Ask AI (Cmd+K) — v1.0

Query entire planning history across all projects.

**Outcome:** Implemented in Phase 6. Project-scoped panel + global modal with cross-project search.

### [x] F7: Goals Management — v1.0

Set and manage goals that influence AI prioritization.

**Outcome:** Implemented in Phase 9. CRUD with goal types (revenue, strategic, quick_win) and tiers.

### [x] F8: Analytics Dashboard — v1.0

Metrics dashboard with funnel visualization.

**Outcome:** Implemented in Phase 11. Period selector, metric cards, funnel visualization, velocity stats.

## API Integrations

### [x] Claude API — v1.0

All conversation interactions, document generation, memory extraction, priority calculations.

**Outcome:** claude-sonnet-4-20250514 for all AI features.

### [x] Perplexity API — v1.0

Deep research during planning phases.

**Outcome:** Implemented in Phase 8 with rate limiting (10/session, 50/project).

### [x] OpenAI Embeddings API — v1.0

Generate embeddings for memories and semantic search.

**Outcome:** text-embedding-3-small (1536 dimensions) with HNSW index.
