## Folder Structure

```
{client-name}/
├── README.md              # This file - client overview
├── contacts.md            # Key contacts at this organization
├── programs/              # Programs delivered to this client
│   └── YYYY-MM-program.md # Individual program records
├── proposals/             # Proposals sent to this client
│   └── YYYY-MM-DD-proposal.md
├── communications/        # Important communication log
│   └── YYYY-MM-DD-topic.md
└── notes.md               # General notes and intelligence
```
