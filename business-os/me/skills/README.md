# Personal Skills

> Custom skills for your personal workflows and routines.

---

## What Are Personal Skills?

Personal skills are instructions you create for recurring tasks in your work. Unlike business skills (used by AI coworkers), these are specifically for tasks where YOU are doing the work with AI assistance.

---

## Structure

```
skills/
├── README.md                    # This file
├── morning-startup.md           # Start your day ritual
├── weekly-review.md             # Personal weekly review
├── content-creation.md          # Writing workflow
├── decision-making.md           # Framework for decisions
├── meeting-prep.md              # Prepare for meetings
└── end-of-day.md                # Close out the day
```

---

## Skill Template

```markdown
# [Skill Name]

**Purpose:** [What this skill helps you accomplish]
**When to Use:** [Trigger or schedule]
**Time Required:** [Estimate]

---

## Context for AI

[Background information Claude needs to help effectively]

## Steps

### Step 1: [Name]
[Instructions]

### Step 2: [Name]
[Instructions]

### Step 3: [Name]
[Instructions]

## Output

[What the end result should look like]

## Notes

[Any additional context or variations]
```

## How to Use Personal Skills

1. **Create the skill** - Write the markdown file with your process
2. **Invoke it** - Reference the skill when you want to run it
3. **Iterate** - Improve the skill based on what works

Example:
> "Let's run my morning-startup skill"

---

## Tips for Good Skills

1. **Be specific** - The more detail, the better the assistance
2. **Include context** - What does Claude need to know?
3. **Define outputs** - What should the result look like?
4. **Note variations** - Are there different modes or situations?
5. **Keep them updated** - Refine as you learn what works

---
