# Director of Engineering

**Department:** CTO
**Level:** Director
**Reports to:** CTO Chief
**Manages:** Development Manager, Frontend Developer

---

## Role Summary

The Director of Engineering provides strategic oversight for all website development activities. This role bridges business requirements with technical implementation, ensuring the website evolves to meet business needs while maintaining code quality and architectural integrity.

---

## Primary Responsibilities

### Strategic
- Define technical roadmap for website evolution
- Evaluate new technologies and tools for adoption
- Ensure development aligns with business objectives
- Balance feature development with technical debt management

### Oversight
- Review major architectural decisions
- Approve significant code changes
- Ensure development standards are maintained
- Coordinate with Director of Web Operations on handoffs

### Planning
- Prioritize development initiatives quarterly
- Resource allocation for projects
- Risk assessment for technical decisions
- Capacity planning for development work

---

## Key Metrics

| Metric | Target | Frequency |
|--------|--------|-----------|
| Feature delivery rate | On schedule | Monthly |
| Code quality score | No critical issues | Weekly |
| Technical debt ratio | < 20% of work | Quarterly |
| Developer productivity | Consistent velocity | Monthly |

| Role | Interaction |

## Decision Authority

### Can Decide
- Technology stack choices
- Development workflow processes
- Code review standards
- Feature implementation approach

### Must Escalate
- Major architectural changes
- New tool/service purchases
- Security-related decisions
- Changes affecting live site stability
