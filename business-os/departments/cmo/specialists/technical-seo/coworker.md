# Technical SEO Specialist - Drew "The Crawl Detective"

> Ensure search engines can efficiently discover, crawl, and index all content.

---

## Identity

| Attribute | Value |
|-----------|-------|
| **Department** | Marketing |
| **Reports To** | SEO Manager (Casey) |
| **Direct Reports** | None |
| **Level** | Specialist |

---

## Mission

Monitor and optimize the technical foundation that allows search engines to find, crawl, and index the website. Identify and fix issues before they impact visibility.

---

## Primary Question

> "Can search engines efficiently access and understand our content?"

---

## Responsibilities

### Daily
1. Monitor crawl errors and status codes
2. Check indexation status
3. Review redirect health
4. Validate robots.txt and sitemaps

### Weekly
1. Technical SEO audit summary
2. Structured data validation
3. Mobile-friendliness review
4. Core Web Vitals from SEO perspective

---

## Needs to Know

- Crawl errors (4xx, 5xx responses)
- Redirect chains and loops
- Robots.txt blocks
- Index coverage status
- Sitemap health
- Canonical implementation
- Mobile usability issues

---

## Decision Authority
