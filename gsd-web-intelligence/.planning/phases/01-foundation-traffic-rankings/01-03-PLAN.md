## Tasks

### Task 1: Build TRF-02 - Traffic Anomaly Detector
**n8n Workflow:** "Web Intel - Traffic - Anomaly Detector"

**Schedule:** Daily at 7:00 AM CT

**Nodes:**
1. Schedule Trigger (7:00 AM CT)
2. Supabase - Query yesterday's traffic
3. Supabase - Query 7-day average (using view)
4. Supabase - Query 30-day average (using view)
5. Code Node - Compare and detect anomalies:
   - Drop >20% from 7-day avg → flag
   - Spike >50% from 7-day avg → flag
   - Calculate severity (critical/warning/info)
6. IF anomaly detected:
   - Create alert record in `web_intel.alerts`
   - Trigger TRF-03 webhook
7. Log success
8. Error Handler

**Anomaly Detection Logic:**
```javascript
const threshold_drop = 0.20;  // 20% drop
const threshold_spike = 0.50; // 50% spike

const change = (today - avg7day) / avg7day;

if (change < -threshold_drop) {
  return { type: 'traffic_drop', severity: change < -0.40 ? 'critical' : 'warning' };
}
if (change > threshold_spike) {
  return { type: 'traffic_spike', severity: 'info' };
}
return null;
```

### Task 2: Build TRF-03 - Traffic Alert Notifier
**n8n Workflow:** "Web Intel - Traffic - Alert Notifier"

**Trigger:** Webhook (from TRF-02)

**Nodes:**
1. Webhook Trigger
2. Code Node - Format Slack message
   - Include: metric, current value, expected value, % change
   - Include: affected pages if applicable
   - Include: link to GA4
3. Slack - Send to #web-intel
4. Update alert record with `notified_at`
5. Log success
6. Error Handler

**Slack Message Format:**
```
🚨 *Traffic Alert: {severity}*

*{alert_type}* detected for {date}

• Current: {current_sessions} sessions
• Expected: {expected_sessions} sessions (7-day avg)
• Change: {percent_change}%

Top affected pages:
{page_list}

<{ga4_link}|View in GA4>
```

### Task 3: Build TRF-06 - Geographic Traffic Analyzer
**n8n Workflow:** "Web Intel - Traffic - Geographic Analyzer"

**Schedule:** Weekly Monday at 6:00 AM CT

**Nodes:**
1. Schedule Trigger (Monday 6:00 AM CT)
2. GA4 API - Fetch by country/region for past 7 days
3. Supabase - Get previous week's geo data
4. Code Node - Compare and identify trends:
   - New countries appearing
   - Countries with significant growth
   - Countries with significant decline
5. Supabase - Store in `web_intel.geo_traffic`
6. IF significant changes → create info alert
7. Log success
8. Error Handler
