# Plan 01-04: Ranking Collection Workflows

## Goal
Build ranking collection workflows using DataForSEO API.

## Wave
4 (Can run in parallel with Wave 3, depends on 01-01 for schema)

## Prerequisites
- 01-01-PLAN.md completed (schema exists)
- DataForSEO credentials registered in n8n-brain
- Initial keyword list prepared (200-500 keywords)

## Tasks

### Task 1: Seed Initial Keyword List
Populate `web_intel.tracked_keywords` with initial keywords.

**Categories:**
- Brand keywords (10-20)
- Program page keywords (100-150)
- Competitor brand terms (20-30)
- Industry terms (50-100)
- Long-tail variations (50-100)

**SQL seed script:** `supabase/scripts/seed-web-intel-keywords.sql`

**Columns:**
- keyword (text)
- search_volume (integer, can be null initially)
- difficulty (integer, can be null)
- priority (high/medium/low)
- target_url (text, expected ranking page)
- category (brand/program/competitor/industry/longtail)

### Task 2: Build RNK-01 - Daily Rank Tracker
**n8n Workflow:** "Web Intel - Rankings - Daily Tracker"

**Schedule:** Daily at 5:00 AM CT

**Nodes:**
1. Schedule Trigger (5:00 AM CT)
2. Supabase - Fetch all tracked keywords
3. Split into batches (50 keywords per batch for API limits)
4. Loop: For each batch:
   - DataForSEO SERP API - Check rankings
   - Parse response for our domain
   - Extract: position, ranking URL, SERP features
5. Supabase - Upsert to `web_intel.daily_rankings`
6. Log success with keyword count
7. Error Handler (with retry logic for rate limits)

**DataForSEO Request:**
```json
{
  "keyword": "{keyword}",
  "location_code": 2840,
  "language_code": "en",
  "device": "desktop",
  "os": "windows"
}
```

**Response Parsing:**
- Find our domain in organic results
- Extract position (1-100 or null if not found)
- Extract ranking URL
- Note SERP features present (featured_snippet, local_pack, etc.)

### Task 3: Build RNK-04 - SERP Feature Tracker
**n8n Workflow:** "Web Intel - Rankings - SERP Features"

**Schedule:** Daily at 5:15 AM CT

**Nodes:**
1. Schedule Trigger (5:15 AM CT)
2. Supabase - Fetch today's ranking data
3. Code Node - Extract SERP features per keyword:
   - Featured snippet present? Who owns it?
   - People Also Ask present?
   - Local pack present?
   - Knowledge panel present?
4. Supabase - Store in `web_intel.serp_features`
5. Code Node - Identify opportunities:
   - We rank top 3 but don't have featured snippet
   - Featured snippet exists and we could target it
6. IF opportunities found → create info alert
7. Log success
8. Error Handler

### Task 4: Build RNK-07 - Search Volume Updater
**n8n Workflow:** "Web Intel - Rankings - Volume Updater"

**Schedule:** Monthly 1st at 3:00 AM CT

**Nodes:**
1. Schedule Trigger (1st of month, 3:00 AM CT)
2. Supabase - Fetch all tracked keywords
3. Split into batches
4. Loop: For each batch:
   - DataForSEO Keyword Data API - Get search volumes
5. Supabase - Update `web_intel.tracked_keywords` with new volumes
6. Code Node - Flag keywords with >50% volume change
7. Log success
8. Error Handler

### Task 5: Register Workflows and Store Patterns

**Register:**
- RNK-01 in n8n-brain workflow registry
- RNK-04 in n8n-brain workflow registry
- RNK-07 in n8n-brain workflow registry

**Store pattern:**
- "DataForSEO SERP Checker" pattern
- "Batch API Processor" pattern
