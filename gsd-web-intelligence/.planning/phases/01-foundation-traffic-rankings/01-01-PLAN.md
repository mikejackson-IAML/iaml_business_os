# Plan 01-01: Database Schema + n8n-brain Setup

## Goal
Create the web_intel database schema in Supabase and register credentials in n8n-brain.

## Wave
1 (Foundation - no dependencies)

## Tasks

### Task 1: Create web_intel Schema
Create the Supabase migration with all Phase 1 tables.

**File:** `supabase/migrations/20260120_create_web_intel_schema.sql`

**Tables to create:**
- `web_intel.daily_traffic` - Aggregate daily traffic metrics
- `web_intel.page_traffic` - Per-page traffic data
- `web_intel.geo_traffic` - Geographic breakdown
- `web_intel.tracked_keywords` - Keywords to monitor
- `web_intel.daily_rankings` - Daily ranking snapshots
- `web_intel.serp_features` - SERP feature tracking
- `web_intel.alerts` - All system alerts
- `web_intel.collection_log` - Track collection runs

**Include:**
- Primary keys, foreign keys, indexes
- Timestamp columns with defaults
- Check constraints where appropriate
- Comments on tables and columns

### Task 2: Create Helper Views
Add views for common queries.

**Views:**
- `web_intel.traffic_7day_avg` - Rolling 7-day traffic average by page
- `web_intel.traffic_30day_avg` - Rolling 30-day traffic average
- `web_intel.ranking_changes` - Today vs yesterday rankings
- `web_intel.ranking_trends` - 7-day ranking trend

### Task 3: Create Helper Functions
Add functions for common operations.

**Functions:**
- `web_intel.log_collection(workflow_id, status, details)` - Log workflow runs
- `web_intel.create_alert(type, severity, title, message, metadata)` - Create alerts
- `web_intel.get_baseline_traffic(page_path, days)` - Get traffic baseline

### Task 4: Register Credentials in n8n-brain
Register all required credentials for Phase 1.

**Credentials to register:**
- `ga4` - GA4 service account
- `dataforseo` - DataForSEO API credentials
- `slack-web-intel` - Slack webhook for #web-intel

### Task 5: Set n8n-brain Preferences
Set workflow naming and style preferences.

**Preferences:**
- `naming.workflow_prefix` = "Web Intel"
- `naming.category_separator` = " - "
- `style.error_handling` = "standard" (try/catch + Slack alert)

## Acceptance Criteria
- [ ] `web_intel` schema exists in Supabase
- [ ] All tables created with proper indexes
- [ ] Views return correct data
- [ ] Functions execute without error
- [ ] Credentials registered in n8n-brain
- [ ] Preferences set in n8n-brain

## Estimated Duration
15-20 minutes
