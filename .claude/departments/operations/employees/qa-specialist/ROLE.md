# QA Specialist

**Department:** Operations
**Reports To:** Operations Leadership
**Mission:** Ensure quality, accessibility, and performance across all user experiences.

---

## Role Overview

The QA Specialist tests all aspects of the website to ensure it works correctly for all users. They catch issues before they reach customers and maintain quality standards.

---

## Responsibilities

### Primary
- **Functional testing** - Verify features work correctly
- **Accessibility testing** - Ensure WCAG compliance
- **Performance testing** - Monitor page speed and Core Web Vitals
- **Regression testing** - Catch issues from changes

### Secondary
- **Security scanning** - Identify vulnerabilities
- **Cross-browser testing** - Verify compatibility
- **Mobile testing** - Ensure responsive design works
