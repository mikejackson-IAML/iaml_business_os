# Email Summary - Send Accomplishment Summary

Generate and send an accomplishment summary email on demand, or preview before sending.

## Usage

```
/email-summary                  # Send today's summary email
/email-summary --preview        # Preview without sending
/email-summary --week           # This week's summary
/email-summary --month          # This month's summary
```

## Workflow

### Step 1: Gather Data

Query accomplishments for the requested period:

```sql
-- For daily summary
SELECT * FROM accomplishments.get_entries_for_period(CURRENT_DATE, CURRENT_DATE);

-- For weekly summary
SELECT * FROM accomplishments.get_entries_for_period(
  date_trunc('week', CURRENT_DATE)::DATE,
  CURRENT_DATE
);
```

Query active goals:

```sql
SELECT * FROM accomplishments.get_active_goals();
```

### Step 3: Preview or Send

**Preview Mode (`--preview`):**

Display the formatted email content without sending:

```
=== EMAIL PREVIEW ===

To: mike.jackson@iaml.com
Subject: Daily Progress: January 14, 2026

[formatted content shown above]

---
[S] Send now
[E] Edit recipient
[C] Cancel
```

**Send Mode (default):**

Trigger the n8n webhook:

```bash
curl -X POST https://n8n.realtyamp.ai/webhook/accomplishment-email \
  -H "Content-Type: application/json" \
  -d '{
    "type": "daily",
    "date": "2026-01-14",
    "recipient": "mike.jackson@iaml.com",
    "entries": [...],
    "goals": [...],
    "trigger": "manual"
  }'
```

### Step 4: Log the Summary

Record in database:

```sql
INSERT INTO accomplishments.email_summaries (
  summary_date,
  summary_type,
  entries_included,
  goals_snapshot,
  email_subject,
  sent_at,
  sent_to,
  status,
  trigger_type
) VALUES (
  CURRENT_DATE,
  'daily',
  ARRAY[...entry_ids...],
  $goals_json,
  'Daily Progress: January 14, 2026',
  NOW(),
  ARRAY['mike.jackson@iaml.com'],
  'sent',
  'manual'
);
```

### Step 5: Confirm

```
=== EMAIL SENT ===

Summary email sent to mike.jackson@iaml.com

Subject: Daily Progress: January 14, 2026
Entries included: 5
Goals tracked: 3

Check your inbox!
```

## Options

| Flag | Description |
|------|-------------|
| `--preview` | Show email content without sending |
| `--week` | Generate weekly summary (Mon-today) |
| `--month` | Generate monthly summary |
| `--to <email>` | Override recipient email |
| `--yesterday` | Send yesterday's summary |

## Email Template Variables

The n8n workflow uses these variables:

| Variable | Description |
|----------|-------------|
| `{{date}}` | Summary date formatted |
| `{{entry_count}}` | Total entries |
| `{{high_impact_count}}` | High/critical entries |
| `{{goals_progressed}}` | Number of goals with progress |
| `{{high_impact}}` | Array of high-impact entries |
| `{{medium_impact}}` | Array of medium-impact entries |
| `{{low_impact}}` | Array of low-impact entries |
| `{{goals}}` | Array of active goals with progress |
| `{{tomorrow_focus}}` | Suggested priorities |

## Configuration

Default recipient is set in the n8n workflow. Override with `--to`:

```
/email-summary --to "mike@personal.com"
```

## Related Commands
