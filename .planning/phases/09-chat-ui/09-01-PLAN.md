  <action>
Create the Chat/Services directory and implement ChatService as an actor (matching NetworkManager pattern):

```swift
actor ChatService {
    private let session = URLSession.shared

    /// Stream chat response from API
    /// - Parameters:
    ///   - messages: Conversation history
    ///   - apiKey: Mobile API key from Keychain
    /// - Returns: AsyncThrowingStream of ChatEvent
    func streamChat(
        messages: [ChatMessage],
        apiKey: String
    ) async throws -> AsyncThrowingStream<ChatEvent, Error>
}
```

Implementation details:
1. Build POST request to `Constants.API.baseURL + "/chat"`
2. Set headers: `X-API-Key`, `Content-Type: application/json`, `Accept: text/event-stream`
3. Encode body as `{"messages": [{"role": "user", "content": "..."}]}`
4. Use `session.bytes(for: request)` to get async byte stream
5. Iterate `bytes.lines` (handles SSE line buffering automatically)
6. Parse lines starting with `data: ` (skip others like empty lines)
7. JSON decode each data payload into ChatEvent
8. Yield events via continuation
9. Finish on `done` event or stream end
10. Handle errors: network failures, bad status codes, decode errors

Add helper struct for request body:
```swift
private struct ChatRequest: Encodable {
    let messages: [[String: String]]
}
```

Map messages using `message.toAPIMessage()` before encoding.
