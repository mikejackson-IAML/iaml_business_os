<task type="auto">
  <name>Task 2: Create chat route with SSE infrastructure</name>
  <files>dashboard/src/app/api/mobile/chat/route.ts</files>
  <action>
Create POST route handler at `dashboard/src/app/api/mobile/chat/route.ts`:

1. **Runtime configuration** (required for streaming):
```typescript
export const runtime = 'nodejs';
export const dynamic = 'force-dynamic';
```

2. **Auth validation** (same pattern as health endpoint):
- Get `X-API-Key` header
- Compare against `process.env.MOBILE_API_KEY`
- Return 401 JSON response if invalid

3. **Request parsing**:
- Parse JSON body expecting `{ messages: Array<{ role: string, content: string }> }`
- Return 400 if messages missing or empty

4. **SSE stream creation**:
- Create `ReadableStream` with `TextEncoder`
- Use `controller.enqueue()` to send events
- Format as SSE: `data: ${JSON.stringify(payload)}\n\n`

5. **Initial implementation** (placeholder for Claude integration):
- For now, echo back a simple response to verify streaming works
- Send events: `{ type: 'text', content: 'Echo: ' + lastMessage }`
- End with: `{ type: 'done', stop_reason: 'end_turn' }`

6. **Response headers**:
```typescript
return new Response(stream, {
  headers: {
    'Content-Type': 'text/event-stream',
    'Cache-Control': 'no-cache, no-transform',
    'Connection': 'keep-alive',
  },
});
```

7. **Error handling**:
- Wrap in try/catch
- On error, send `{ type: 'error', message: string }` then close stream
- Log errors to console

Import Anthropic SDK but don't use it yet - just verify it imports cleanly:
```typescript
import Anthropic from '@anthropic-ai/sdk';
// Will use in next plan
```
  </action>
  <verify>
