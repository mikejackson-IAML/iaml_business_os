  <action>
Create view modifier following RESEARCH.md pattern:

1. `ToastModifier`:
   - @Binding var toast: Toast?
   - Uses .overlay(alignment: .bottom) to position toast
   - Applies .transition(.move(edge: .bottom).combined(with: .opacity))
   - Adds padding(.bottom, 100) to clear tab bar
   - Uses .animation(.spring(duration: 0.3), value: toast)
   - On appear of toast, schedule Task to dismiss after 2 seconds:
     ```swift
     Task {
         try? await Task.sleep(for: .seconds(2))
         withAnimation { toast = nil }
     }
     ```

2. View extension:
   ```swift
   extension View {
       func toast(_ toast: Binding<Toast?>) -> some View {
           modifier(ToastModifier(toast: toast))
       }
   }
   ```

Create Shared/Modifiers/ directory if it doesn't exist.
