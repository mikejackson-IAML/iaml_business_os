Create ViewModel following HomeViewModel pattern:

1. `@MainActor class QuickActionsViewModel: ObservableObject`:
   - `@Published var actions: [QuickAction] = []`
   - `@Published var isLoading = false`
   - `@Published var loadingActionId: String? = nil` (which button is loading)
   - `@Published var toast: Toast? = nil`
   - `@Published var confirmationAction: QuickAction? = nil` (for risky actions)

2. `loadActions(context: LAContext) async`:
   - Set isLoading = true
   - Call NetworkManager.shared.fetchWorkflows(context:)
   - Filter to first 6 actions (grid limit per CONTEXT.md)
   - Set actions = result.workflows
   - Set isLoading = false
   - On error: show error toast

3. `triggerAction(_ action: QuickAction, context: LAContext) async`:
   - If action.riskLevel != .safe: set confirmationAction = action, return (wait for confirm)
   - Call performTrigger(action, context:)

4. `confirmAndTrigger(context: LAContext) async`:
   - Guard let action = confirmationAction else return
   - confirmationAction = nil
   - await performTrigger(action, context:)

5. `private func performTrigger(_ action: QuickAction, context: LAContext) async`:
   - Set loadingActionId = action.id
   - Call NetworkManager.shared.triggerWorkflow(workflowId: action.id, parameters: nil, context:)
   - On success: toast = Toast(message: "Sent!", type: .success)
   - On failure: toast = Toast(message: "Failed: \(error.userMessage)", type: .error)
   - Set loadingActionId = nil

Use HapticManager.shared.tap() when button pressed.

**QuickActionsGrid.swift:**

Create 2x3 grid following RESEARCH.md pattern:

1. Define columns: `[GridItem(.flexible()), GridItem(.flexible())]`

2. Grid structure:
   ```swift
   LazyVGrid(columns: columns, spacing: 12) {
       ForEach(viewModel.actions) { action in
           QuickActionButton(
               action: action,
               isLoading: viewModel.loadingActionId == action.id,
               onTap: { Task { await viewModel.triggerAction(action, context: context) } }
           )
       }
   }
   ```

3. QuickActionButton (can be nested struct or separate):
   - VStack with icon + label
   - ProgressView when isLoading, else Image(systemName: action.icon)
   - Icon color: .safe = .blue, .risky = .orange, .destructive = .red
   - Frame height 80, maxWidth .infinity
   - Background: Color(.systemBackground)
   - Corner radius from Constants.UI.cornerRadius
   - ButtonStyle .plain with scaleEffect on press

4. Add confirmation alert:
   ```swift
   .alert("Run \(viewModel.confirmationAction?.name ?? "Workflow")?",
          isPresented: .constant(viewModel.confirmationAction != nil)) {
       Button("Cancel", role: .cancel) { viewModel.confirmationAction = nil }
       Button("Run") { Task { await viewModel.confirmAndTrigger(context: context) } }
   } message: {
       Text("This action may make changes.")
   }
   ```

5. Apply toast modifier: `.toast($viewModel.toast)`
