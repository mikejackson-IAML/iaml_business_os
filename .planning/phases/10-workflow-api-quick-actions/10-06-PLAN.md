  <action>
Create settings view for quick action configuration:

1. **State management:**
   - `@AppStorage("enabledActionIds")` var enabledActionIdsData: Data = Data()
   - Helper computed property to encode/decode [String] array
   - `@State var availableActions: [QuickAction] = []`
   - `@State var enabledActions: [QuickAction] = []` (ordered subset)
   - `@Environment(AppState.self) var appState`

2. **Load actions on appear:**
   - Fetch all workflows from NetworkManager
   - Separate into enabled (in order from AppStorage) and disabled lists
   - Maximum 6 enabled actions per CONTEXT.md

3. **UI Structure:**
   ```swift
   List {
       Section("Enabled (drag to reorder)") {
           ForEach($enabledActions) { $action in
               ActionRow(action: action, isEnabled: true) {
                   removeAction(action)
               }
           }
           .onMove(perform: moveAction)
       }

       Section("Available") {
           ForEach(disabledActions) { action in
               ActionRow(action: action, isEnabled: false) {
                   addAction(action)
               }
           }
       }
   }
   .environment(\.editMode, .constant(.active)) // Always show drag handles
   ```

4. **ActionRow component:**
   - HStack: icon, name, description (if any)
   - Toggle or +/- button to enable/disable
   - Disabled if enabled count is 6 and trying to add

5. **Persistence:**
   - On any change, encode enabledActions ids to Data
   - Store in @AppStorage

6. **Navigation bar:**
   - Title: "Quick Actions"
   - Done button if presented modally

Create Shared/Extensions/AppStorage+Codable.swift if needed for Data<->[String] encoding.
