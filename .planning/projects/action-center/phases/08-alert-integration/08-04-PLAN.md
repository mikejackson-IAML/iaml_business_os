# Plan 08-04: AI Transformation Node (Claude Integration)

## Objective

Add Claude AI integration to transform alert titles into action-oriented task titles and generate concise task descriptions. This replaces the placeholder in 08-03 with actual AI transformation.

## Context

From 08-CONTEXT.md:
- Title format: Action-oriented (transform "SSL Certificate Expiring Soon" → "Renew SSL Certificate")
- Transformation approach: AI transformation via Claude
- Description content: AI-generated actionable summary focused on what to do

From 08-RESEARCH.md:
- Existing pattern: HeyReach uses HTTP request to call AI API (Gemini)
- For Claude: Use HTTP request node to Anthropic API

## Tasks

<task id="1">
Create Claude transformation HTTP request node:

**Node: `transform-with-claude`**
- Type: n8n-nodes-base.httpRequest
- Method: POST
- URL: `https://api.anthropic.com/v1/messages`
- Headers:
  - `x-api-key`: `{{ $credentials.anthropicApi.apiKey }}`
  - `anthropic-version`: `2023-06-01`
  - `content-type`: `application/json`

**Request Body:**
```json
{
  "model": "claude-3-haiku-20240307",
  "max_tokens": 500,
  "messages": [
    {
      "role": "user",
      "content": "Transform this alert into an actionable task.\n\nAlert Type: {{ $json.alert_type }}\nAlert Title: {{ $json.title }}\nAlert Description: {{ $json.description }}\nAffected Resource: {{ $json.affected_resource }}\n\nRespond with JSON only:\n{\n  \"task_title\": \"Action-oriented title starting with a verb (e.g., 'Renew SSL Certificate', 'Investigate Domain Health Issue')\",\n  \"task_description\": \"2-3 sentence actionable summary of what needs to be done and key context. Do not include the full alert details, just the essential action items.\"\n}"
    }
  ]
}
```

**Response Handling:**
- Parse JSON from `content[0].text`
- Extract `task_title` and `task_description`
</task>

<task id="2" depends_on="1">
Create fallback transformation for when AI fails:

**Node: `fallback-transform`**
- Type: n8n-nodes-base.set
- Executes when Claude request fails or returns invalid JSON

**Fallback Logic:**
```javascript
// Generate basic action-oriented title
const alertType = $json.alert_type;
const title = $json.title;
const resource = $json.affected_resource;

// Simple transformation patterns
const actionVerbs = {
  'ssl_expiry': 'Renew',
  'domain_health': 'Investigate',
  'uptime_down': 'Restore',
  'payment_failed': 'Resolve',
  'tier_ending': 'Address',
  'vip_non_response': 'Follow up on'
};

const verb = actionVerbs[alertType] || 'Address';
const taskTitle = `${verb}: ${title}`;
const taskDescription = `${$json.description}\n\nAffected: ${resource}`;

return {
  task_title: taskTitle,
  task_description: taskDescription
};
```
</task>
