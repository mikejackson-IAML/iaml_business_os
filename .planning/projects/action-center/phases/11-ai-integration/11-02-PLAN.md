# Plan 11-02: Weekly Focus n8n Workflow

## Goal
Create n8n workflow that runs Sunday evening (planning) and Friday afternoon (recap) to generate AI analysis and store Weekly Focus Review task.

## Requirements Covered
- AI-01: Weekly AI Focus generation (Sunday evening + Friday recap)
- AI-03: Creates "Weekly Focus Review" task with full analysis

## Context
- Pattern: Follow daily-digest-sender.json structure
- Schedule: Sunday 7pm CT (planning), Friday 5pm CT (recap)
- Workflow calls /api/action-center/ai-analysis endpoint
- Creates task with task_type='review', source='ai'

## Tasks

<task autonomous="true">
<title>Create Weekly AI Focus workflow JSON</title>
<file path="business-os/workflows/weekly-ai-focus.json" action="create" />
<description>
n8n workflow structure:

1. Schedule Trigger
   - name: "Weekly Planning and Recap"
   - Two schedules: "0 19 * * 0" (Sunday 7pm), "0 17 * * 5" (Friday 5pm)
   - Timezone: America/Chicago

2. Determine Mode (Code node)
   - Check day of week from trigger
   - Set mode = "planning" for Sunday, "recap" for Friday
   - Pass mode to next node

3. Call AI Analysis API (HTTP Request)
   - POST to {{$env.DASHBOARD_URL}}/api/action-center/ai-analysis
   - Body: { "mode": {{$json.mode}}, "max_suggestions": 10 }
   - HTTP Header Auth credential for API key
   - Timeout: 120000ms (Claude can take time)

4. Check for Errors (IF node)
   - If $json.success !== true, route to error handler

5. Create Weekly Focus Task (Supabase Insert)
   - Table: action_center.tasks
   - title: "Weekly Focus Review - {{$now.format('MMM D, YYYY')}}"
   - description: Build markdown from analysis:
     ## This Week's Focus
     {{$json.data.summary}}

     ## Suggested Actions
     {{loop over suggestions}}

     ## Patterns Noticed
     {{loop over patterns}}

     ## Last Week Review
     {{$json.data.last_week_review}}
   - task_type: 'review'
   - source: 'ai'
   - priority: 'normal'
   - due_date: NULL (review task, no due date)
   - dedupe_key: "weekly_focus:{{$now.format('YYYY-WW')}}"

6. Create AI Suggestions (Loop)
   - For each suggestion in $json.data.suggestions:
   - Insert into action_center.tasks with source='ai', ai_confidence, ai_suggested_at

7. Slack Success Log
   - Channel: #ai-focus
   - Message: Weekly AI Focus generated for week {{week}}
   - Include suggestion count

8. Error Handler (Slack alert to #alerts)
   - Message: Weekly AI Focus generation failed
   - Include error details

Connections:
- Schedule -> Determine Mode -> Call API -> Check Errors
- Check Errors (false) -> Create Task -> Create Suggestions -> Slack Success
- Check Errors (true) -> Error Handler
</description>
</task>
