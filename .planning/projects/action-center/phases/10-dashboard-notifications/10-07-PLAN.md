# Plan 10-07: Resend Email Service and Digest Template

## Objective

Create the daily digest email template using React Email components and a function to send the digest via Resend.

## Context

The daily digest email should have a friendly, conversational tone:
- Greeting: "Hey [Name], here's what's on your plate today..."
- Critical section (if any): Red highlight, task titles
- Due Today section (if any): Amber highlight, task titles
- Overdue section (if any): Red urgent highlight, task titles
- Quick stats: Total tasks, completed this week
- CTA: "View Action Center" button

Resend supports React components for email templates.

<tasks>
<task id="1">
Install @react-email/components for email template building
</task>
<task id="2">
Create email template component at `dashboard/src/lib/email/templates/daily-digest.tsx`:
- Props: recipientName, criticalTasks, dueTodayTasks, overdueTasks, stats
- Friendly conversational header
- Conditional sections for critical/due today/overdue (only show if count > 0)
- Task list items with titles and due dates
- Stats footer
- "View Action Center" CTA button
</task>
<task id="3">
Style email with inline CSS for email client compatibility:
- Red (#DC2626) for critical/overdue sections
- Amber (#D97706) for due today section
- Clean, professional layout
- Mobile-responsive width
</task>
<task id="4">
Create send function at `dashboard/src/lib/email/send-digest.ts`:
- Takes userId and optional overrides
- Fetches user profile (name, email, notification prefs)
- Fetches task data for critical, due today, overdue
- Renders template and sends via Resend
- Returns success/error result
</task>
<task id="5">
Add "From" email configuration: "IAML Action Center <actioncenter@yourdomain.com>"
</task>
<task id="6">
Handle edge case: Skip sending if nothing urgent (no critical, due today, or overdue)
</task>
</tasks>
