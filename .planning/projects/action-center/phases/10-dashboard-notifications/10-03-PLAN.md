# Plan 10-03: Action Center Dashboard Widget Component

## Objective

Create the Action Center widget component that displays task counts (Critical, Due Today, Overdue) with clickable chips that navigate to filtered task views.

## Context

The widget will be placed on the main CEO dashboard. It should display:
- Critical count (red chip with alert icon)
- Due Today count (amber chip with clock icon)
- Overdue count (red chip with attention icon) - only shown if count > 0
- "View all X tasks" link at bottom

Clicking a chip navigates to `/dashboard/action-center?filter=critical|due_today|overdue`.

The task count data will come from the RPC created in 10-01.

<tasks>
<task id="1">
Add server-side function `getTaskCounts()` to `dashboard/src/lib/api/task-queries.ts` that calls `action_center.get_task_counts()` RPC
</task>
<task id="2">
Create widget component at `dashboard/src/components/widgets/action-center-widget.tsx` with:
- Card container using existing Card component from dashboard-kit
- Header with "Action Center" title and CheckSquare icon
- Three count chips in a row: Critical (red), Due Today (amber), Overdue (red, conditional)
- Each chip shows icon + count + label
- "View all X tasks" link at bottom
</task>

</task>
