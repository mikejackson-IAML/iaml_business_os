# Plan 10-02: Install Dependencies - Resend and Sonner

## Objective

Install Resend SDK for transactional email and Sonner for toast notifications in the dashboard project.

## Context

The dashboard uses Next.js 16.1.1 with React 19. We need:
- `resend` - Official Resend SDK for sending transactional emails (daily digest, critical alerts)
- `sonner` - Toast notification library that works well with React 19 and provides a clean API

Neither dependency is currently installed per package.json review.

<tasks>
<task id="1">
Install resend package: `npm install resend`
</task>
<task id="2">
Install sonner package: `npm install sonner`
</task>
<task id="3">
Add Toaster component to root layout at `dashboard/src/app/layout.tsx` inside the body, after ThemeProvider
</task>
<task id="4">
Create Resend client utility at `dashboard/src/lib/email/resend.ts` that exports a configured Resend instance using RESEND_API_KEY env var
</task>
<task id="5">
Add RESEND_API_KEY to `.env.example` and document in README if exists
</task>
</tasks>
