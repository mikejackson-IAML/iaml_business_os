## Goal

Allow reopening completed tasks within 7 days of completion.

## Requirements Covered

- Success criteria #6 (completed task reopening within 7 days)

## Context

Tasks have `completed_at` timestamp when marked done. User should be able to reopen a task if they realize it wasn't actually complete, but only within a reasonable window (7 days).

## Tasks

<task id="1">
Add `reopenTask` mutation to `task-mutations.ts`:

```typescript
export async function reopenTask(taskId: string): Promise<void> {
  const supabase = await createClient();

  // Verify task is done and within 7 days
  const { data: task, error: fetchError } = await supabase
    .from('tasks')
    .select('id, status, completed_at')
    .eq('id', taskId)
    .single();

  if (fetchError) throw fetchError;
  if (task.status !== 'done') throw new Error('Task is not completed');

  const completedAt = new Date(task.completed_at);
  const daysSinceCompletion = differenceInDays(new Date(), completedAt);
  if (daysSinceCompletion > 7) {
    throw new Error('Task cannot be reopened after 7 days');
  }

  // Reopen: set status to 'open', clear completed_at
  const { error: updateError } = await supabase
    .from('tasks')
    .update({
      status: 'open',
      completed_at: null,
      updated_at: new Date().toISOString()
    })
    .eq('id', taskId);

  if (updateError) throw updateError;
}
```

export async function reopenTaskAction(taskId: string): Promise<ActionResult> {
  try {
    await reopenTask(taskId);

    // Log activity
    await logTaskActivity(taskId, 'status_changed', {
      from: 'done',
      to: 'open',
      action: 'reopened'
    });

    revalidatePath(`/dashboard/action-center/tasks/${taskId}`);
    revalidatePath('/dashboard/action-center');

    return { success: true };
  } catch (error) {
    return { success: false, error: error.message };
  }
}

Add helper function `canReopenTask(task)`:

```typescript
export function canReopenTask(task: TaskExtended): boolean {
  if (task.status !== 'done') return false;
  if (!task.completed_at) return false;

  const completedAt = new Date(task.completed_at);
  const daysSinceCompletion = differenceInDays(new Date(), completedAt);

  return daysSinceCompletion <= 7;
}
```
