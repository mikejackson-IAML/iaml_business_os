## Goal

Add personal stats widget to CEO dashboard showing completion metrics and weekly goal progress.

## Requirements Covered

- MET-01 (personal stats on dashboard)

## Context

User wants stats displayed in BOTH places: dashboard widget AND task list banner. This plan creates the dashboard widget.

The widget should fit alongside existing widgets (WeeklyFocusWidget, ActionCenterWidget).

<task id="2">
Implement widget with card layout showing:
- **Header**: "Your Progress" with trophy or chart icon
