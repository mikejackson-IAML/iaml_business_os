## Goal

Add `task_preferences` JSONB column to profiles table for storing weekly goal settings and create helper functions.

## Requirements Covered

- MET-01 (weekly goal storage)

## Context

The user wants weekly goal gamification for personal stats. This requires storing the user's target in a preferences column on their profile.

**Preferences structure:**
```json
{
  "weekly_goal_target": 15,
  "weekly_goal_enabled": true
}
```

## Tasks

<task id="1">
Create migration file `supabase/migrations/20260126_task_preferences_metrics.sql`

Add `task_preferences` JSONB column to `public.profiles`:
```sql
ALTER TABLE public.profiles
ADD COLUMN IF NOT EXISTS task_preferences JSONB DEFAULT '{}';

COMMENT ON COLUMN public.profiles.task_preferences IS 'User preferences for Action Center: {weekly_goal_target, weekly_goal_enabled}';
```
</task>

<task id="2">
Create `action_center.get_user_task_preferences(p_user_id UUID)` function that returns the task_preferences JSONB from profiles, defaulting to empty object if null.
</task>

<task id="3">
Create `action_center.set_user_weekly_goal(p_user_id UUID, p_target INTEGER)` function that:
- Updates profiles.task_preferences to set weekly_goal_target and weekly_goal_enabled=true
- Uses jsonb_set to preserve other preferences
- Returns the updated preferences
</task>

<task id="4">
Create `action_center.get_completed_today_count(p_user_id UUID)` function that:
- Counts tasks with status='done' and completed_at >= CURRENT_DATE
- Filters by assignee_id = p_user_id
- Returns integer count
</task>

<task id="5">
Create `action_center.get_weekly_goal_progress(p_user_id UUID)` function that:
- Gets completed_this_week from user_task_summary view
- Gets weekly_goal_target from profiles.task_preferences
- Returns JSONB with: completed, target, progress_percent
- Returns null for progress_percent if no goal set
</task>
