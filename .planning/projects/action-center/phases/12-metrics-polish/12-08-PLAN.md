## Goal

Integrate task metrics into overall department health score calculation.

## Requirements Covered

- MET-04 (task metrics feed into department health scores)

## Context

The existing mobile health API calculates scores for departments (Digital, Workflows). Task metrics should contribute to the overall health picture.

**Health Score Formula:**
- Base score: 100
- Completion rate (30d): contributes 50% of score
- Overdue penalty: up to 30 points deducted based on overdue % of actionable
- Critical resolution speed: up to 20 points for handling critical tasks quickly

**Integration approach:** Add "Tasks" as a new department in the health response, and factor it into overall health.

<task id="2">
Implement score calculation:
```typescript
// Base: 100 points
let score = 100;

// Completion rate contribution (50 points max)
// 80%+ completion = full 50 points, scales down linearly
const completionContribution = Math.min(50, (completion_rate_30d / 80) * 50);
score = completionContribution + 50; // Start with completion contribution

// Overdue penalty (up to 30 points)
// 0% overdue = no penalty, 20%+ overdue = full 30 point penalty
const overduePercent = overdue_count / Math.max(actionable_count, 1) * 100;
const overduePenalty = Math.min(30, overduePercent * 1.5);
score -= overduePenalty;

// Critical bonus/penalty (up to 20 points)
// No open critical = +20, each open critical = -5
const criticalImpact = Math.max(-20, 20 - (critical_count * 5));
score += criticalImpact;

// Clamp to 0-100
score = Math.max(0, Math.min(100, score));
```
</task>
