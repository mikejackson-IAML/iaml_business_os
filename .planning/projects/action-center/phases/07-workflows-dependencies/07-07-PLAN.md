# Plan 07-07: Workflow Task List with Dependencies

## Goal
Build the task list component for workflow detail that shows tasks in dependency order with visual indentation for dependent tasks.

## Context
Per 07-CONTEXT: Tasks shown as ordered list with dependency indentation. Dependent tasks indented with arrow icon showing what blocks them. Blocked tasks: muted opacity + warning badge. Clicking task navigates to task detail page.

## Tasks

<task id="1">
Create `workflow-task-row.tsx` with:
- Link wrapper to task detail page
- Indentation level prop (0, 1, 2) for visual hierarchy
- Status icon (CheckCircle=done, Clock=in_progress/waiting, Circle=open)
- Task title
- Priority indicator (colored dot)
- Due date with overdue highlighting
- Blocked badge when is_blocked=true (muted text + warning badge)
- Blocked-by indicator showing parent task name when indented
</task>

<task id="2">
Create dependency sorting function:
```typescript
function sortTasksByDependency(tasks: TaskExtended[]): { task: TaskExtended; depth: number }[]
```
- Build dependency graph from depends_on arrays
- Topological sort to order tasks
- Calculate depth level (0=no dependencies, 1+=has dependencies)
- Handle circular dependencies gracefully (log warning, flatten)
</task>

<task id="3">
Create `workflow-task-list.tsx` with:
- Section header "Tasks in this Workflow"
- Card container
- Sorts tasks using dependency sorting function
- Maps over sorted tasks rendering WorkflowTaskRow with depth
- Border-left with increasing margin for depth levels
- Empty state if no tasks
</task>

<task id="4">
Add visual styling for blocked tasks:
- Muted text color (text-muted-foreground)
- Warning badge: "Blocked by N" with amber styling
- Slightly reduced opacity (opacity-80)
</task>

<task id="5">
Integrate into workflow-detail-content.tsx:
- Import WorkflowTaskList
- Pass workflow.tasks to component
- Remove placeholder section
</task>

<task id="6">
Add exports to components/index.ts:
- export { WorkflowTaskList } from "./workflow-task-list";
- export { WorkflowTaskRow } from "./workflow-task-row";
</task>
