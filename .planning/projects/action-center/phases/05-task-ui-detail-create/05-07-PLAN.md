# Plan 05-07: Activity Tab

## Objective
Create the activity tab component displaying task history/changes.

## Tasks

<task id="1">
Create `task-activity.tsx` with imports:

- Activity, CheckCircle, XCircle, Clock, AlertTriangle, MessageSquare, User, Workflow from lucide-react
- Card, CardContent from dashboard-kit
- TaskActivity type
- formatDistanceToNow from date-fns

Define activityConfig mapping activity_type to { icon, color, label }:
- created: Activity, text-blue-500, 'Task created'
- status_changed: Clock, text-amber-500, 'Status changed'
- completed: CheckCircle, text-emerald-500, 'Completed'
- dismissed: XCircle, text-gray-500, 'Dismissed'
- priority_changed: AlertTriangle, text-orange-500, 'Priority changed'
- comment_added: MessageSquare, text-blue-500, 'Comment added'
- assigned: User, text-purple-500, 'Assigned'
- workflow_changed: Workflow, text-cyan-500, 'Workflow updated'
- approved: CheckCircle, text-emerald-500, 'Approved'
- rejected: XCircle, text-red-500, 'Rejected'
- modified: Activity, text-amber-500, 'Modified & Approved'
</task>

<task id="2">
Implement ActivityItem component:

- Get config from activityConfig (with fallback for unknown types)
- Display icon with config color in rounded background
- Show label and timestamp (formatDistanceToNow with addSuffix)
- Build description based on activity_type:
  - status_changed/priority_changed: show old_value → new_value
  - Others: show new_value if exists
- Show actor name or actor_type or 'System'
</task>

<task id="3">
Implement main TaskActivityList component:

Props:
- activity: TaskActivity[]

Layout:
- Card wrapper with CardContent p-4
- Empty state: Activity icon, "No activity recorded"
- Activity list: divide-y divide-border, each ActivityItem
</task>
