# Plan 05-10: Workflow Context Component

## Objective
Create the workflow context component showing progress and sibling tasks when a task belongs to a workflow.

## Tasks

<task id="1">
Create `workflow-context.tsx` with imports:

- useState from 'react'
- Link from 'next/link'
- ChevronDown, ChevronRight, Workflow, CheckCircle, Circle, Clock from lucide-react
- Card, CardContent, CardHeader, CardTitle from dashboard-kit
- Progress from dashboard-kit
- TaskExtended type
</task>

<task id="2">
Implement WorkflowContext component:

Props:
- task: TaskExtended
- workflowTasks?: TaskExtended[] (optional, for future sibling task display)

State:
- isExpanded: boolean (for collapsible task list)

Behavior:
- Return null if !task.workflow_id
- Calculate progress from workflowTasks if provided
</task>

<task id="3">
Implement the component layout:

Card structure:
- CardHeader with Workflow icon and "Part of Workflow" title
- CardContent with:

1. Workflow name link:
   - Link to `/dashboard/action-center/workflows/${task.workflow_id}`
   - Show task.workflow_name or 'View Workflow'
   - Status badge if task.workflow_status exists

2. Progress bar (if workflowTasks provided):
   - "Progress" label with "X of Y complete"
   - Progress component with calculated percentage

3. Collapsible task list (if workflowTasks provided):
   - Toggle button with ChevronDown/ChevronRight
   - List of tasks with status icons:
     - done: CheckCircle (emerald)
     - in_progress: Clock (amber)
     - other: Circle (muted)
   - Link to each task detail page
   - Highlight current task (font-medium)

4. Note (if no workflowTasks):
   - "View the full workflow for detailed progress and all related tasks."
</task>
