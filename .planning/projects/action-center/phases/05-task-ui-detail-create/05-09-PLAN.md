# Plan 05-09: Approval Task UI

## Objective
Create the approval task UI with recommendation display and Approve/Modify/Reject buttons.

## Tasks

<task id="1">
Create `approval-actions.tsx` with imports:

- useState, useTransition from 'react'
- ThumbsUp, ThumbsDown, Edit2, X, Lightbulb from lucide-react
- Button from dashboard-kit
- Card, CardContent, CardHeader, CardTitle from dashboard-kit
- TaskExtended type
- approveTaskAction from actions
</task>

<task id="2">
Implement RecommendationCallout component (exported):

Props: { task: TaskExtended }

Behavior:
- Return null if task.task_type !== 'approval' or !task.recommendation
- Render Card with amber left border, amber background tint
- Lightbulb icon
- "Recommendation" heading
- task.recommendation content
- If task.recommendation_reasoning: show reasoning in border-t section

This satisfies APPR-01: Approval task type shows recommendation and reasoning.
</task>

<task id="3">
Implement ModifyApproveDialog component (internal):

Props:
- taskId: string
- isOpen: boolean
- onClose: () => void

State:
- modifications: string (required)
- isPending: from useTransition
- error: string | null

Behavior:
- Modal with Edit2 icon and "Modify & Approve" title
- Textarea for modifications (required)
- Validate modifications not empty
- Call approveTaskAction(taskId, 'modified', modifications)
- Close on success

This satisfies APPR-03: Modification requires details entry.
</task>

<task id="4">
Implement main ApprovalActions component (exported):

Props: { task: TaskExtended }

Behavior:
- If task.approval_outcome exists: show decision badge (approved=green, rejected=red, modified=amber)
- Otherwise show three buttons:
  1. Approve (emerald bg): calls approveTaskAction(id, 'approved', null)
  2. Modify & Approve (outline): opens ModifyApproveDialog
  3. Reject (outline with red text): calls approveTaskAction(id, 'rejected', null)

State:
- showModifyDialog: boolean
- isPending: from useTransition
- error: string | null

This satisfies APPR-02: Approval actions: Approve, Modify & Approve, Reject.
</task>
