# Plan 05-05: Complete and Dismiss Dialogs

## Objective
Create dialog components for completing tasks (with optional note) and dismissing tasks (with required reason).

## Tasks

<task id="1">
Create `complete-task-dialog.tsx` following the modal pattern from `assign-instructor-modal.tsx`:

Props interface:
- taskId: string
- taskTitle: string
- isOpen: boolean
- onClose: () => void

State:
- note: string (optional completion note)
- isPending: from useTransition
- error: string | null

Behavior:
- Fixed positioning overlay with backdrop
- Card-based modal body with CardHeader and CardContent
- Textarea for optional note
- Cancel and "Complete Task" buttons
- Call completeTaskAction on submit
- Close on success, show error on failure
- Click backdrop to close

This satisfies UI-13: Complete button with completion note prompt.
</task>

<task id="2">
Create `dismiss-task-dialog.tsx` with required reason dropdown:

Define dismissReasons array:
- { value: 'no_longer_relevant', label: 'No Longer Relevant' }
- { value: 'duplicate', label: 'Duplicate Task' }
- { value: 'will_not_do', label: 'Will Not Do' }
- { value: 'other', label: 'Other' }

Props interface:
- taskId: string
- taskTitle: string
- isOpen: boolean
- onClose: () => void

State:
- reason: string (required)
- notes: string (optional additional context)
- isPending: from useTransition
- error: string | null

Behavior:
- Fixed positioning overlay with backdrop
- Card-based modal body
- Select dropdown for reason (required, validated before submit)
- Textarea for optional notes
- Cancel and "Dismiss Task" (destructive variant) buttons
- Validate reason is selected before submit
- Call dismissTaskAction on submit
- Close on success, show error on failure

This satisfies UI-14: Dismiss button with required reason prompt.
</task>
