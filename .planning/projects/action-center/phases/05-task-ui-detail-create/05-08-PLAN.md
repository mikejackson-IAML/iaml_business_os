Create the task creation modal with all form fields (title required, others optional).

## Tasks

<task id="1">
Create `create-task-modal.tsx` with imports:

- useState, useTransition from 'react'
- X, Plus from lucide-react
- Button from dashboard-kit
- Card, CardContent, CardHeader, CardTitle from dashboard-kit
- createTaskAction from actions
- TaskType, TaskPriority types

Define constants:
- taskTypes: standard, approval, decision, review
- priorities: critical, high, normal, low
</task>

<task id="2">
Implement form state and submission:

Props:
- isOpen: boolean
- onClose: () => void
- departments: string[]

State:
- title: string (required)
- description: string
- taskType: TaskType (default 'standard')
- priority: TaskPriority (default 'normal')
- dueDate: string
- department: string
- isPending: from useTransition
- error: string | null

Functions:
- resetForm: clear all fields and error
- handleSubmit: validate title, call createTaskAction, reset and close on success
- handleClose: reset and call onClose
</task>

<task id="3">
Implement the form UI:

Layout:
- Fixed positioning overlay with backdrop (click to close)
- Card with max-w-lg, max-h-[90vh], overflow-hidden
- CardHeader with Plus icon and "Create Task" title, X close button
- CardContent with overflow-y-auto and form

Form fields:
1. Title (required): text input with autoFocus
2. Description: textarea, 3-4 rows
3. Two-column grid: Type (select) + Priority (select)
4. Two-column grid: Due Date (date input) + Department (select with departments prop)

Error display: red background div with error message

Actions: Cancel (outline) and Create Task buttons, border-t separator
</task>
