Create a client component that displays ranking position changes with visual indicators.

Props:
- `change: number | null` - The position change value (positive = dropped, negative = improved, null/0 = no change)
- `className?: string` - Optional additional classes

Logic:
- If change is null or 0: Show a dash character in muted text
- If change < 0 (improvement - position went from 10 to 5 = -5):
  - Show green text with ArrowUp icon
  - Display as "+{absValue}" (e.g., "+5" meaning 5 positions better)
- If change > 0 (drop - position went from 5 to 10 = +5):
  - Show red text with ArrowDown icon
  - Display as "-{absValue}" (e.g., "-5" meaning 5 positions worse)
- If change >= 10 (dramatic drop):
  - Additionally show AlertTriangle icon before the arrow

Use:
- lucide-react for icons (ArrowUp, ArrowDown, Minus, AlertTriangle)
- Tailwind classes: text-success for green, text-error for red, text-muted-foreground for dash
- cn utility from @/dashboard-kit/lib/utils

Example outputs:
- change=-5 → green "↑ +5"
- change=3 → red "↓ -3"
- change=12 → red "⚠️ ↓ -12"
- change=0 → gray "—"
