<task type="auto">
  <name>Task 2: Create Supabase query functions</name>
  <files>dashboard/src/lib/api/web-intel-queries.ts</files>
  <action>
Add query functions after the interfaces:

**getDailyTraffic(days: number = 30):**
- Query web_intel.daily_traffic
- Order by collected_date DESC
- Limit to `days` records
- Return DailyTrafficDb[]

**getTopPages(days: number = 7, limit: number = 10):**
- Query web_intel.page_traffic
- Filter where collected_date >= now - days
- Order by pageviews DESC
- Limit results
- Return PageTrafficDb[]

**getTrackedKeywords(status: string = 'active'):**
- Query web_intel.tracked_keywords
- Filter by status
- Order by priority (critical, high, medium, low), then keyword
- Return TrackedKeywordDb[]

**getDailyRankings(days: number = 7):**
- Query web_intel.daily_rankings
- Filter where collected_date >= now - days
- Order by collected_date DESC
- Return DailyRankingDb[]

**getRankingChanges(days: number = 7, significantOnly: boolean = false):**
- Query web_intel.ranking_change_events
- Filter where detected_date >= now - days
- If significantOnly, filter is_significant = true
- Order by detected_date DESC, change_amount DESC
- Return RankingChangeEventDb[]

**getCoreWebVitals():**
- Query web_intel.core_web_vitals
- Get most recent for each device_type
- Order by collected_date DESC
- Limit 2 (mobile + desktop)
- Return CoreWebVitalsDb[]

**getIndexCoverage():**
- Query web_intel.index_coverage
- Get single most recent record
- Return IndexCoverageDb | null

**getSearchPerformance(days: number = 7, limit: number = 20):**
- Query web_intel.search_performance
- Filter where collected_date >= now - days
- Order by clicks DESC
- Return SearchPerformanceDb[]

**getContentDecay(unresolvedOnly: boolean = true):**
- Query web_intel.content_decay
- If unresolvedOnly, filter is_addressed = false
- Order by decay_percentage DESC
- Return ContentDecayDb[]

**getBacklinkProfile():**
- Query web_intel.backlink_profile
- Get single most recent record
- Return BacklinkProfileDb | null

**getWebIntelAlerts(unresolvedOnly: boolean = true):**
- Query web_intel.alerts
- If unresolvedOnly, filter acknowledged_at IS NULL
- Order by severity (critical, warning, info), then created_at DESC
- Return WebIntelAlertDb[]

Each function should:
1. Call getServerClient()
2. Build query with proper filters
3. Handle errors with console.error and return empty array/null
4. Cast result to proper type

Use schema prefix for all tables: `.from('web_intel.daily_traffic')` etc.
