# Plan 04-02: Core Web Vitals Card Component

## Objective

Create the unified CoreWebVitalsCard that displays all three CWV metrics (LCP, INP, CLS) with a mobile/desktop toggle and overall status badge.

## Requirements Covered

- CWV-01: LCP display with status (via CwvMetric)
- CWV-02: CLS display with status (via CwvMetric)
- CWV-03: INP display with status (via CwvMetric)
- CWV-04: Mobile/Desktop toggle (uses DeviceToggle from 04-01)
- CWV-05: Overall CWV status ("Passing" or "Needs Work")
