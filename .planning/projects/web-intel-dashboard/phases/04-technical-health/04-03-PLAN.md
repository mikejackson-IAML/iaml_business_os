Aggregation logic (from RESEARCH.md):
```typescript
function aggregateGscMetrics(data: SearchPerformance[]) {
  const totalClicks = data.reduce((sum, d) => sum + d.clicks, 0);
  const totalImpressions = data.reduce((sum, d) => sum + d.impressions, 0);

  const ctr = totalImpressions > 0
    ? (totalClicks / totalImpressions) * 100
    : 0;

  // Weighted average position (more accurate than simple average)
  const weightedPosition = totalImpressions > 0
    ? data.reduce((sum, d) => sum + (d.position ?? 0) * d.impressions, 0) / totalImpressions
    : 0;

  return { totalClicks, totalImpressions, ctr, avgPosition: weightedPosition };
}
```

Layout: Grid of 4 MetricCards (same pattern as TrafficMetricsRow line 119)
```tsx

  <MetricCard label="CTR" value={metrics.ctr} format="percent" icon={Percent} />
  <MetricCard label="Avg Position" value={metrics.avgPosition} format="number" icon={TrendingUp} description="Lower is better" />
</div>
```

<task id="2">
Create `top-queries-list.tsx` in `dashboard/src/app/dashboard/web-intel/components/`.

Component specifications:
- Props: `searchPerformance: SearchPerformance[]`, `limit?: number` (default 10)
- Import `SearchPerformance` from `@/lib/api/web-intel-queries`
- Import `Card, CardHeader, CardTitle, CardContent` from `@/dashboard-kit/components/ui/card`

Layout (per CONTEXT.md - simple list format):
```tsx
<Card>
  <CardHeader>
    <CardTitle>Top Queries</CardTitle>
  </CardHeader>
  <CardContent>
    {searchPerformance.length === 0 ? (
      <p className="text-muted-foreground text-center py-4">
        No search query data available.
      </p>
    ) : (
      <div className="space-y-2">
        {searchPerformance.slice(0, limit).map((sp, i) => (
          <div key={sp.id} className="flex items-center justify-between py-2 border-b last:border-0">
            <div className="flex items-center gap-2">
              <span className="text-muted-foreground w-6">{i + 1}.</span>
              <span className="font-medium truncate max-w-[300px]">'{sp.query || '(not set)'}'</span>
            </div>
            <span className="text-muted-foreground">{sp.clicks.toLocaleString()} clicks</span>
          </div>
        ))}
      </div>
    )}
  </CardContent>
</Card>
```

Note: Data is already sorted by clicks descending from `getSearchPerformance()` query.
</task>
