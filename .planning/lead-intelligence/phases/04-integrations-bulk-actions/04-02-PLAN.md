  <action>
    Create campaigns/route.ts (GET):
    - validateApiKey, then fetch `https://server.smartlead.ai/api/v1/campaigns?api_key=${process.env.SMARTLEAD_API_KEY}`
    - Filter response to status === 'STARTED' only
    - Return array of { id, name, status, created_at }
    - On SmartLead error, return 502 with { error, code: 'EXTERNAL_API_ERROR' }

    Create contacts/[id]/add-to-campaign/route.ts (POST):
    - validateApiKey, accept body: { campaignId: number }
    - Fetch contact from Supabase by ID to get email, first_name, last_name, company
    - POST to `https://server.smartlead.ai/api/v1/campaigns/${campaignId}/leads?api_key=${key}`
    - Body: `{ lead_list: [{ email, first_name, last_name, company_name }] }`
    - Return { success: true, added: 1 } or error

    Create bulk/add-to-campaign/route.ts (POST):
    - validateApiKey, accept body: { contactIds: string[], campaignId: number }
    - Fetch all contacts by IDs from Supabase
    - Check for duplicates: GET campaign leads from SmartLead, compare emails
    - If duplicates found, return { duplicates: [...], newContacts: [...] } with status 200 and `action: 'confirm_needed'`
    - If no duplicates (or second call with `confirmed: true`), POST leads in batches of 10 with 500ms delay between batches
    - Return { success: true, added: N, skipped: N }
  </action>
