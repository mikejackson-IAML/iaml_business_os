**Queries file:**
- `getContacts(params: ContactListParams)` — uses getServerClient(), selects `*, companies(name, id)`, applies `.range()` for pagination, `.order()` for sorting, returns `{ count: 'exact' }`. Returns ContactListResponse.
- `getContactById(id: string)` — selects `*, companies(name, id)`, `.eq('id', id)`, `.single()`. Returns Contact or null.

**Mutations file:**
- `createContact(input: CreateContactInput)` — inserts, `.select().single()`, returns Contact.
- `updateContact(id: string, input: UpdateContactInput)` — updates `.eq('id', id)`, `.select().single()`, returns Contact or null.
- `deleteContact(id: string)` — deletes `.eq('id', id)`, returns boolean success.

**List route (contacts/route.ts):**
- GET: Parse query params (page, limit, sort, order), call getContacts, return JSON.
- POST: Parse JSON body, validateCreateContact, call createContact, return 201.
- Both: validateApiKey(request) first. Wrap in try/catch with structured error responses.

**Detail route (contacts/[id]/route.ts):**
- GET: await context.params for id, call getContactById, return 404 if null.
- PUT: await context.params, parse body, validateUpdateContact, call updateContact, return 404 if null.
- DELETE: await context.params, call deleteContact, return 204 on success.
- All: validateApiKey first. Use `interface RouteContext { params: Promise<{ id: string }> }` (Next.js 16 async params).

Follow the existing task API route pattern exactly. Import validateApiKey from wherever the task routes import it.
